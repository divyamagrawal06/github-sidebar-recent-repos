# GitHub Sidebar Recent Repos Extension

Chrome extension that adds a recent repositories section to GitHub's left sidebar. Made this because top repositories always felt useless to me. Anyway, that's the extension.

## Install from GitHub Releases

1. Go to this repository's [Releases](../../releases) page.
2. Download the latest release ZIP.
3. Extract the ZIP to a folder on your machine.
4. Open `chrome://extensions`. (Or manage extensions in other browsers.)
5. Enable **Developer mode**. Toggle will be on top right of screen most likely.
6. Click **Load unpacked**.
7. Select the folder you just extracted (NOT the ZIP file.)

## Settings

You can click the extension icon in the toolbar to open a popup and configure how many repos to display. Default is 10.
