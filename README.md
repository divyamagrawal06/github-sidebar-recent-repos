# GitHub Sidebar Recent Repos Extension

Chrome extension that adds a recent repositories section to GitHub's left sidebar. Made this because top repositories always felt useless to me. Anyway, that's the extension.

## Installation

### Chrome or any Chromium based browsers

1. Go to the [latest release](https://github.com/divyamagrawal06/github-sidebar-recent-repos/releases/tag/1.0.0).
2. Download the latest release ZIP tagged **`-chromium`**.
3. Unzip the **`-chromium`** release asset.
4. Open `chrome://extensions` (or `edge://extensions`, etc.).
5. Turn on **Developer mode**.
6. **Load unpacked** and choose the unzipped folder. 
7. Ensure that `manifest.json` is in the folder you select, not in a nested folder.

### Firefox

Note: This is not permanent, firefox extensions require being signed with firefox to be permanent. You will have to do this every time you open firefox, which isn't ideal.

1. Go to the [latest release](https://github.com/divyamagrawal06/github-sidebar-recent-repos/releases/tag/1.0.0).
2. Download the latest release ZIP tagged **`-firefox`**.
3. Open `about:debugging#/runtime/this-firefox`.
4. Under **Temporary Extensions**, click **Load Temporary Add-on…** and select the zip file.

## Settings

Use the toolbar icon to set how many repos appear in the sidebar (1–50; default 10).
